# html-github-kodutoo1

Static multi-page website for the NordClean home and office cleaning service concept. The site is built with HTML and Bootstrap and includes a landing page, services overview, and contact form.

## Pages

- `index.html` - Landing page with the main value proposition and call-to-action buttons.
- `services.html` - Services overview with cards for window cleaning, home cleaning, office cleaning, and add-on services.
- `contact.html` - Contact form and service area details.

## Features

- Responsive navigation with Bootstrap 5.
- Consistent brand styling via custom CSS variables and Google Fonts.
- Service cards with images and short descriptions.
- Contact form with required fields and consent checkbox.

## Tech stack

- HTML5
- Bootstrap 5.3.3 (CDN)
- Google Fonts (Fraunces, Manrope)

## Assets

Images are stored under `images/` and referenced by the services page cards.

## Run locally

Open `index.html` in a browser. Use the navigation to access the services and contact pages.
